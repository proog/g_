About g_

g_ is a web application for displaying game collections.
It supports multiple users, and all games, platforms, genres and tags are entirely editable.

g_ uses the following third-party components:

  • AngularJS
  • Angular Loading Bar
  • Angular Upload
  • AngularUI Bootstrap
  • Bootstrap
  • Chart.js
  • Illuminate Database
  • Moment.js
  • Slim Framework
